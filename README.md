**horribleAttention**

- **Project:** A small experimental collection exploring attention mechanisms, demos, and tooling.
- **Purpose:** Fast, hacky prototypes and visualizations for attention-related ideas â€” not production-ready.

**Highlights**
- **Interactive demo:** A Streamlit demo is included to visualize and play with attention components.
- **Standalone visualization:** `attention.html` contains a static demo you can open in a browser.
- **Multi-language examples:** Contains small Python and C++ experiments (`main.py`, `main.cc`).

**Quick Start**
- Install Python dependencies (recommended):

```
pip install -r requirements-streamlit.txt
```

- Run the Streamlit demo:

```
streamlit run streamlit_app.py
```

- Or open the static demo: open `attention.html` in your browser.

**Repository Layout**
- **attention.html:** Static attention visualization/demo.
- **streamlit_app.py:** Streamlit demo UI and examples.

**Goals & Philosophy**

- Prefer clarity and quick iteration over optimization or robustness.
